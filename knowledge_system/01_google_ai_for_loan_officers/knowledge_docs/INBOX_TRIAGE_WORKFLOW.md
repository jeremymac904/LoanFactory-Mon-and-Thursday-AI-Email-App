# INBOX_TRIAGE_WORKFLOW

## Purpose

Outline a first pass AI-assisted inbox triage workflow for loan officers who need a faster first response block.

## Source Basis

1. `01_google_ai_for_loan_officers/01_NOTEBOOK_BRIEF.md`
2. `01_google_ai_for_loan_officers/04_DOMAIN_MAP.md`

## Current Status

First pass workflow built for internal training use.

## Needs Factual Validation

1. Exact inbox tools and channels used by the target team
2. Whether CRM note sync is manual or partially automated today

## Needs Compliance Review

1. Any draft reply that references approvals, rates, products, fees, or timelines

## Workflow

1. Pull the top active borrower, realtor, and ops threads first
2. Group by urgency and response type
3. Draft only the first response pass with AI
4. Review sensitive language before send
5. Log the next action in the CRM or task system
