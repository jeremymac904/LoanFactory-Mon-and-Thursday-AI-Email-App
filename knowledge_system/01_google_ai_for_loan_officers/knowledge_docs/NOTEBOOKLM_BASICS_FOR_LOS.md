## Basic Flow

1. Load a clean source set for one problem only
2. Ask NotebookLM to summarize, compare, or extract actions from those sources
3. Use the output to identify one lesson or one decision support angle
4. Move the usable result into a draft, prompt asset, or archive note

## Best Uses In This Product

1. Transcript to one usable lesson
2. Guideline lookup or stress testing
3. Topic bank creation
4. Audio overview and recap for internal learning

## What To Avoid

1. Treating NotebookLM output like approved policy
2. Mixing too many unrelated sources into one notebook
3. Reusing sensitive file detail without review
