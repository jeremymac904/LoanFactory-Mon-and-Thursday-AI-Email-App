## Core Positioning Rules

1. Explain the value of choice and structure without hype.
2. Use plain language instead of giant broker-industry claims.
3. Tell the reader what is known, what is not known, and what the next step is.
4. Avoid certainty when the fact base is weak.

## Strong Positioning

1. `We help compare options and structure the next move clearly.`
2. `We do not assume the same solution fits every borrower or file.`
3. `We will confirm the specifics before anyone treats them like settled truth.`

## Weak Positioning

1. Unverified best-price or guarantee claims
2. Unsupported lender-count claims
3. Fee claims without validation
4. Any statement that sounds like policy when it is really a draft talking point

## Training Email Angles

1. Transparency when the file slows down
2. Explain broker value without sounding salesy
3. Trust language that reduces anxiety
4. Safe before-and-after rewrite examples

## Needs Factual Validation

1. Company facts used in public or training positioning
2. Any fee amounts, lender counts, guarantee mechanics, or vendor capability claims

## Needs Compliance Review

1. Any live positioning statement used externally
2. Any borrower-facing or realtor-facing copy derived from this playbook
