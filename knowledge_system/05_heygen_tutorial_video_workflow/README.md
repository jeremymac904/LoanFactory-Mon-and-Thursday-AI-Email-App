This lane supports the first-pass HeyGen tutorial video workflow for the internal Monday and Thursday training email studio.

## What This Lane Covers

1. Script approval before render
2. Structured avatar video generation only
3. Polling-first render tracking for V1
4. Final human review before attach

2. Multi-scene or B-roll workflows
3. Translation flows

5. Webhooks
