# Script Generation Prompts

Use the approved email draft as the source of truth.

## V1 Prompt Goals

1. Convert the approved draft into a short tutorial script
2. Preserve the approved meaning
3. Keep the script compatible with structured avatar rendering
4. Keep compliance-sensitive lines explicit and visible

## Prompt Constraints

1. Do not invent unsupported company facts
2. Do not rewrite the message into a new marketing angle
3. Do not remove review language from sensitive topics
4. Keep the script easy for text-to-speech playback

## Suggested Output Shape

1. One short opener
2. Two to four tactical teaching lines
3. One short close
4. One compliance footer placeholder that stays flagged for review

## Needs Factual Validation

1. Whether HeyGen supports any reliable tone-control or emotion tags for this exact setup

## Needs Compliance Review

1. Any live compliance footer text
2. Any script prompt that could remove required review language from regulated topics
