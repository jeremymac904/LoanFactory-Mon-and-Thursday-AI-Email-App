# Loan Factory Codex Source Packets

This folder is the upload ready source packet bundle for Codex.

Purpose

These files convert the NotebookLM and Gemini outputs that existed only in chat into clean markdown source packets that Codex can import into the repo and turn into real knowledge docs, topic banks, prompt libraries, and app behavior improvements.

What is included

1. Weekly Training Production Lab packet
2. Sales, Marketing, Communication, and Conversion packet
3. Loan Factory and TerraPlus Operations packet
4. Google AI for Loan Officers packet
5. Operator normalization notes
6. A Codex import prompt

How to use

1. Upload this zip to Codex in the repo chat
2. Tell Codex to import these packets into a raw source packet area in the repo
3. Tell Codex to use them to upgrade the scaffold knowledge docs and tighten the training email studio
4. Keep all uncertain facts marked as Needs factual validation or Needs compliance review

Important

These packets are working source inputs, not final approved policy.
Anything involving rates, fees, product eligibility, disclosures, borrower data, or compliance must remain flagged for review until validated.

See `05_operator_notes/CODEX_SOURCE_PACKET_IMPORT_PROMPT.md`
