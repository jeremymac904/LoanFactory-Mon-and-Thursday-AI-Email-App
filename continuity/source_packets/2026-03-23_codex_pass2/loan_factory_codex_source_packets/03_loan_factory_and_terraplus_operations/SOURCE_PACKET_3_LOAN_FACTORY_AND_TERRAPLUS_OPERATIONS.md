## Revised 20 Email Ideas

### Immediate Use Priority

#### Monday
**Subject:** Stop manually texting, set up automated rate alerts in MOSO  
**Why this matters:** Passive pipeline management loses deals.  
**The play:** Set target alerts so warm prospects get notified when the market hits their number.  
**Do this today:** Configure alerts for three warm prospects.  
**Optional asset:** Rate drop text template.  
**Optional demo:** Short screen recording showing the setup path.  
**Needs factual validation:** Confirm whether AI drafts transaction specific pricing alerts automatically.  
**Needs compliance review:** Rate and disclosure handling.

#### Monday
**Subject:** Share your digital vCard to capture weekend open house leads  
**Why this matters:** Instant contact sharing beats paper cards and slow follow up.  
**The play:** Use the app QR code so agents and buyers can save you instantly.  
**Do this today:** Save your QR code to your phone favorites.  
**Optional asset:** Short open house text.  
**Optional demo:** 15 second phone recording of generating the QR code.

#### Thursday
**Subject:** Execute a seamless handoff to the internal processor  
**Why this matters:** Sloppy handoffs create delays and missed timelines.  
**The play:** Use a pre submission checklist before sending the file.  
**Do this today:** Keep the handoff checklist pinned for the next file.  
**Optional asset:** Four point handoff checklist.  
**Optional demo:** Click path for a clean handoff.  
**Needs compliance review:** Processor fee and disclosure handling if referenced.

#### Thursday
**Subject:** Catch missing signatures before the processor sees them  
**Why this matters:** Incomplete uploads create avoidable delays.  
**The play:** Use an AI assisted document pass or a checklist before submit.  
**Do this today:** Pre screen the next upload packet.  
**Optional asset:** Missing page or signature nudge text.  
**Optional demo:** Visual of a missing signature being flagged.  
**Needs factual validation:** Exact UI steps for document checking.

#### Thursday
**Subject:** The 2 property issues that will kill your appraisal  
**Why this matters:** Basic safety or habitability misses can blow up the timeline.  
**The play:** Ask about safety detectors and visible condition issues before ordering the appraisal.  
**Do this today:** Verify detector status on the newest contract.  
**Optional asset:** Agent text for pre appraisal check.  
**Optional demo:** Ordering path for appraisal workflow.

#### Thursday
**Subject:** Avoid the Worst Case policy on expiring rate locks  
**Why this matters:** Expired locks can create higher costs or pricing pain.  
**The play:** Watch files with near term expirations and communicate early.  
**Do this today:** Sort the pipeline by lock expiration date.  
**Optional asset:** Lock expiration warning text.  
**Optional demo:** Filter pipeline by expiration date.  
**Needs factual validation:** Specific lender worst case triggers.  
**Needs compliance review:** Lock extension or fee disclosures.

#### Thursday
**Subject:** Draft the perfect 7 day escalation ticket  
**Why this matters:** Vague escalation requests waste time.  
**The play:** Use a standard format with address, delay reason, lock date, and proposed fix.  
**Do this today:** Save the template into your notes app.  
**Optional asset:** Escalation ticket template.  
**Optional demo:** Where to submit it.

#### Thursday
**Subject:** Never execute a rate lock without borrower consent  
**Why this matters:** Locks without clear consent create compliance and borrower issues.  
**The play:** Have the direct conversation and keep a clear trail before locking.  
**Do this today:** Review active floating files and confirm documentation before lock.  
**Optional asset:** Simple borrower consent message.  
**Optional demo:** Documenting consent in the system.  
**Needs compliance review:** Lock authorization requirements.
