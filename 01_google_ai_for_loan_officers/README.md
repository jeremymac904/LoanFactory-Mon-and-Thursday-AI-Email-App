# Google AI for Loan Officers

This folder is a starter kit for one NotebookLM notebook inside Jeremy's loan officer AI training product.

# Project Context

This notebook supports a larger internal product and training system for Loan Factory loan officers.

The larger goal is to help loan officers use AI better, communicate better, create better content, save time, improve workflow, and close more business.

This notebook is not being built as a random knowledge bucket.
It is being built as a scoped source grounded research and synthesis layer for a future internal platform.

That platform is expected to support:

1. Weekly internal training emails
2. Team and company level content variants
3. Searchable knowledge documents
4. Shared Gemini workflows
5. Future portal content
6. Workflow guides, templates, and prompt libraries

## How to use

1. Create a fresh NotebookLM notebook with the same name as this folder.
2. Upload or paste the brief and related files first.
3. Add your best source material for this notebook only.
4. Start with the starter prompt.
5. Use the refinement prompts after the notebook gives you its first grounded output.
6. Turn the strongest answers into new markdown knowledge documents for the product.
