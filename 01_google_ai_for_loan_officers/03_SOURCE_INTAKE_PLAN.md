# SOURCE INTAKE PLAN

## Intake Goal

Build a clean, focused source set for Google AI for Loan Officers.

## Intake Rules

1. Only add sources that clearly belong in this notebook.
2. Prefer fewer high value sources over a giant messy pile.
3. Add internal markdown files first so the notebook has scoped direction.
4. Add official documentation or primary sources before commentary.
5. Add real workflow examples where possible.
6. Remove duplicate or low value sources if they muddy the notebook.

## Source Priority Order
1. Official Google Help and product documentation
2. Google Workspace product pages
3. Google Labs and AI feature explainers
4. Internal markdown files created for this project
5. Examples of real loan officer use cases and training notes
6. Video transcripts or webinars about Google AI workflows

## Intake Workflow

1. Add the notebook brief first.
2. Add the research mission.
3. Add the domain map.
4. Add the output standards.
5. Add the best primary sources.
6. Add the best real world examples.
7. Run the starter prompt.
8. Log source gaps and missing documents.
9. Build new markdown files from the best findings.
